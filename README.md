## Modo de uso da aplicação

- Selecione a opção "Alfabeto".
- Digite no campo de texto "Entrada" a sua sentença para o alfabeto.
- Selecione o botão "Adicionar".
- Selecione a opção "Tokens".
- Digite no campo de texto "Entrada" a sua sentença para a validação. OBS: Para cada "Espaço" a sua sequência será validada e apresentada com uma mensagem de estar correta ou não.
- Caso o usuário queira trocar a sentença do alfabeto é só clicar no botão com ícone de "lixeira" e refazer os passos da explicação.
